# Лабораторная работа №6
1. Создайте публичный репозиторий на GitHub с названием pb-2025-labs
2. Клонируйте созданный репозиторий на компьютер
3. Создайте структуру репозитория:
4. README.md - описание вашего репозитория
5. labs - папка с лабораторными работами
6. labs/lab1 - папка с файлами вашей первой лабораторной работы
7. labs/lab2 - папка для вашей второй лабораторной работы
8. ... - аналогичные папки для остальных лабораторных работ
9. Для первой лабораторной работы создайте файл README.md с описанием (скопировать условия задания)
10. Сделайте коммит
11. Проделайте то же самое для остальных лабораторных работ. Для каждого файла делайте отдельный коммит
12. Создайте новую ветку add-picture и переключитесь на нее
13. В папку с шестой лабораторной работой добавьте любую картинку (цензурную) и сделайте коммит
14. Вернитесь на главную ветку main
15. В папку с шестой лабораторной работой добавьте файл picture.md и напишите в нем краткое описание вашей картинки
16. Сделайте merge commit из add-picture в main
17. Отправьте изменения в репозиторий, если не делали этого ранее
